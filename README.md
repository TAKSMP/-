# ちょうむしアプリ 🐛

撮った虫の写真をよみこむと、**AIが虫を判定**してくれる、こども向けの虫あつめアプリです。
判定した虫は **図鑑** にどんどんきろくされていきます。

## 📱 公開サイト（スマホでそのまま遊べます）

👉 **https://taksmp.github.io/-/**

`main` ブランチに変更をpushすると、GitHub Actions が自動でビルドして
上のURLに公開しなおします（`.github/workflows/deploy.yml`）。
HTTPS配信なので、スマホでもカメラ機能がそのまま使えます。

## できること

- 🔎 **しらべる（ホーム）**
  **📸 カメラでとる**（その場でカメラを立ち上げてさつえい→そのまま判定）か、
  **🖼️ 写真をえらぶ**（すでにある写真をよみこむ）でAIにみせると、
  以下の4つを判定します。
  - 🏷️ **なまえ**
  - 🐾 **目（もく）**（コウチュウ目・チョウ目 など）
  - ⭐ **レア度**（星5だんかい）
  - 🌳 **生息地**（くさむら など）

  判定がまちがっていたら「✏️ まちがいをなおす」でその場でなおせます。
  **📍 みつけたばしょ**（じっさいに見つけた場所）も入力できます。手入力のほか、
  これまでに入れた場所が候補として出てくるのでえらぶだけでもOK。
  「📖 図鑑にきろく！」をおすと図鑑にたまっていきます。

- 🧠 **クイズ**
  **じぶんが図鑑にあつめた虫だけ**から出題される、10もんのテスト（ランダム）。
  自分でとった写真を見て「なまえ・目・生息地」をあてるほか、**せつめい文を
  読んで どの虫かをあてる**問題も出ます（説明文に名前が入っていたら伏せ字に）。
  選択肢も発見ずみの虫からつくられます（虫が2しゅるい以上あつまると あそべます）。
  れんぞく正解でもりあがり、さいごに「虫はかせ」ランクがでます。

- 📖 **ずかん**
  最初に **「もくじ」** が出て、昆虫図鑑の **目（もく）30分類**（『学研の図鑑LIVE
  昆虫』もくじ準拠）がならびます。各項目の横に **発見数**（0なら0）が出て、
  **発見数の多い順**（同数なら本の順番）にならびます。項目をえらぶと、その目に
  分類された虫のページになります。
  おなじ虫を もういちど とると、その虫のページに **とった日と写真が りれき**
  としてたまり、どの写真を **メイン画像** にするかもえらべます。

- 🔍 **けんさく**
  なまえ・目・生息地・レア度でしぼりこみ。
  それぞれの虫の「豆ちしき」も読めます。

### こどもがたのしいしかけ

- 🎉 みつけたときの **紙吹雪** と **ファンファーレ音**
- 🔬 AIが「かんがえている」スキャン演出
- ⭐ レア度がわかる星ひょうか、レア虫（星5）はキラキラ
- 🏆 図鑑コンプリートでおいわい表示
- まるっこくてカラフルな、こども向けデザイン

## うごかしかた

```bash
npm install
npm run dev
```

ブラウザで表示される URL（例: `http://localhost:5173`）をひらいてください。
スマホやタブレットのカメラからそのまま撮影して読みこめます。

図鑑のデータはブラウザ（localStorage）にほぞんされるので、
とじても消えません。

## AIのモードについて

このアプリには2つの判定モードがあります。

### 1. デモモード（キーなしでも遊べる／かんたん）

写真の色を解析して、いちばん近い虫を図鑑からえらぶだけの「当てずっぽう」です。
本物の画像認識ではないので、正確ではありません（緑の虫はぜんぶ緑の虫に寄る等）。
すぐ試したいときや、あそび用です。

### 2. 本物AIモード（おすすめ・正確）

[Claude](https://www.anthropic.com/) の画像認識をつかい、写真の虫を**種レベルの和名**
（例: オオカマキリ、ギンヤンマ）まで正確に判定します。目・レア度・生息地・説明文も
その種に合ったものを返します。昆虫にくわしい子ども向けの精度をめざしています。

**つかいかた（公開サイトでもローカルでも同じ）:**

1. 右上の **⚙️ ボタン**（AIせってい）をひらく
2. [console.anthropic.com](https://console.anthropic.com/settings/keys) で作った
   Anthropic APIキー（`sk-ant-…`）を貼りつけて「ほぞんして有効化」

- キーは**その端末のブラウザ（localStorage）だけ**に保存され、どこにも送信・共有
  されません。リポジトリにも含まれません。
- ブラウザから直接 Anthropic API を呼ぶため、キーは自分専用にしてください。
- 使った分だけ Anthropic から料金がかかります。
- 本物AIモードで判定に失敗したときは、まちがった当てずっぽうを出さずにエラー表示します。

ローカル開発では `.env` に `VITE_ANTHROPIC_API_KEY=…` を書いて既定キーにすることも
できます（任意）。

### 3. AIチャット取り込みモード（APIキーなしでOK・手作業）

お手もちの **AIチャット（ChatGPT / Claude など）** をつかって同定し、その答えを
取り込めます。APIキーは不要で、追加のAPI料金もかかりません。

判定結果の画面で「🤖 AIチャットでもっと正確にしらべる」をひらき、

1. **① 写真と質問をAIチャットへ** … スマホの共有機能で、写真と決まった形式の質問文を
   AIチャットアプリに送ります（共有先に出ないアプリもあります。共有が使えない端末では
   質問文をコピーしてChatGPTを開きます）。
2. AIが `名前: / 目: / レア度: / 生息地: / 説明:` の形で、**コードブロック（```）に入れて**
   答えます。→ チャットアプリの**コピーボタンで全文をワンタップコピー**できます。
3. **貼り付け → ③ 取り込む** を押すと、各項目に自動入力されます（```の囲みが付いたままでもOK）。

※ ChatGPT Plusのような「チャットの月額課金」は、アプリから直接よび出すAPIとは別物です。
このモードは“手作業でAIチャットを使い、答えだけ取り込む”しくみなので、月額課金を活かせます。

## つかっている技術

- [Vite](https://vite.dev/) + [React](https://react.dev/) + TypeScript
- 図鑑の保存: ブラウザの localStorage
- 効果音: Web Audio API（音声ファイルなし）
- 画像なし・絵文字ベースなので、そのまま動きます
