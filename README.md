# ちょうむしアプリ 🐛

撮った虫の写真をよみこむと、**AIが虫を判定**してくれる、こども向けの虫あつめアプリです。
判定した虫は **図鑑** にどんどんきろくされていきます。

## 📱 公開サイト（スマホでそのまま遊べます）

👉 **https://taksmp.github.io/-/**

`main` ブランチに変更をpushすると、GitHub Actions が自動でビルドして
上のURLに公開しなおします（`.github/workflows/deploy.yml`）。
HTTPS配信なので、スマホでもカメラ機能がそのまま使えます。

## できること

- 🔎 **しらべる（ホーム）**
  **📸 カメラでとる**（その場でカメラを立ち上げてさつえい）か、
  **🖼️ 写真をえらぶ**（すでにある写真をよみこむ）で写真をとりこむと、
  そのまま入力フォームがひらきます（自動のAI診断はしません）。ここに、
  - 🏷️ **なまえ**
  - 🐾 **目（もく）**（コウチュウ目・チョウ目 など・30分類からプルダウン）
  - ⭐ **レア度**（星5だんかい）
  - 🌳 **生息地**（くさむら など）

  を入れていきます。**なまえ**は、**「🤖 AIにきく（写真）」**ボタンをおすと
  **写真をコピーした状態で ChatGPT がひらく**ので、入力らんに写真をはって
  「この虫の名前は？」ときき、こたえを なまえ欄に入れられます。
  **生息地**と**📝 せつめい**も、**「🤖 AIにきく（コピー）」**ボタンで質問文が
  コピーされ、AIチャット（ChatGPT / Claude など）に貼りつけて答えをもらい、
  その答えを欄に貼りつけて入力できます（APIキーは不要）。もちろん手入力もできます。
  **📅 みつけた日**（さいしょは きょう。写真に**撮影日(EXIF)**があれば、その日が
  自動で入ります）と **📍 みつけたばしょ**（手入力のほか、これまでの場所を
  **プルダウンからえらぶ**こともできます）も入力できます。
  ばしょは これまでに入れた場所が候補に出てくるので えらぶだけでもOK。
  「📖 図鑑にきろく！」をおすと図鑑にたまっていきます。
  なまえ・目・レア度・生息地・せつめいまで まとめて聞きたいときは
  **「🤖 AIでまとめてしらべる」** から、順番に：
  **① 写真をコピー** → **② ChatGPTをひらく（入力らんに 写真をペースト）** →
  **③ しつもんをコピー（ChatGPTにペーストして送信）**。
  クリップボードは1つしか持てないため、**写真をはってから しつもんをコピー**する
  順番にしてあります。答えを ⑤で取り込みます（APIキー不要）。
  作業中に ずかん・けんさく・クイズのタブへ うっかり移っても、図鑑に登録する
  までは 入力した内容が消えずに のこります。

- 🎮 **あそぶ**
  あつめた虫であそべる、**クイズ**と**バトル**の2つのゲームがえらべます。

  - 🧠 **クイズ** …
    **じぶんが図鑑にあつめた虫だけ**から出題される、10もんのテスト（ランダム）。
    自分でとった写真を見て「なまえ・目・生息地」をあてるほか、**せつめい文を
    読んで どの虫かをあてる**問題も出ます（説明文に名前が入っていたら伏せ字に）。
    選択肢も発見ずみの虫からつくられます。れんぞく正解でもりあがり、さいごに
    「虫はかせ」ランクがでます。

  - ⚔️ **バトル** …
    記録した虫どうしで たたかう たいせんゲーム（虫が2ひき以上で あそべます）。
    各虫に **バトルステータス**（**たいりょく**1〜20のよこゲージ・**こうげき**と
    **ぼうぎょ**は⭐10だんかい・その虫ならではの **ひっさつわざ**）があります。
    ひっさつわざの効果は「つよいいちげき／にかいこうげき／かいふく／こうげきアップ
    ／ぼうぎょアップ」から選べ、**つよい技は1回だけ**つかえる（バランス調整）。
    あそびかたは、**①じぶんの虫をえらぶ →②あいて（ランダム or じぶんでえらぶ）→
    ③じゃんけんで先攻をきめる →④こうげき／ひっさつわざ で たいりょくをけずりあう**。
    こうげき力とぼうぎょ力の差でダメージがきまり、たまに **かいひ**（よけ）も
    でます。ステータスは、図鑑の詳細 **「✏️ ないようを なおす」→「⚔️ バトル
    ステータス」** から、**「🤖 AIにきめてもらう」**でバランスよく決めてもらうか、
    手で調整できます（未設定の虫は レア度から自動でステータスがついて すぐ遊べます）。

- 📖 **ずかん**
  最初に **「もくじ」** が出て、昆虫図鑑の **目（もく）30分類**（『学研の図鑑LIVE
  昆虫』もくじ準拠）がならびます。各項目の横に **発見数**（0なら0）が出て、
  **発見数の多い順**（同数なら本の順番）にならびます。項目をえらぶと、その目に
  分類された虫のページになります。
  おなじ虫を もういちど とると、その虫のページに **とった日と写真が りれき**
  としてたまり、どの写真を **メイン画像** にするかもえらべます。
  図鑑に入れたあとでも、詳細の **「✏️ ないようを なおす」** から、なまえ・目・
  レア度・生息地・みつけたばしょ・せつめい を **あとから修正**できます
  （生息地・せつめいは「🤖 AIにきく（コピー）」で質問をコピー→AIチャットの
  答えを貼りつけ）。**とった日づけは、履歴の写真ごとに 個別になおせます**。

- 🎯 **ミッション**
  虫さがしが もっと たのしくなる おだいページ。**Easy（かんたん）/ Normal（ふつう）
  / Hard（むずかしい）** の3レベルから、いつも **3つのミッションが同時に**でます
  （1レベル＝1つ）。おだいの れい：「ちがう虫を◯しゅるい みつけよう」「◯つの
  目（もく）を あつめよう」「◯つの ばしょで みつけよう」「レアな虫を みつけよう」
  「おなじ虫を 2かい とろう」など。進み具合は 図鑑データから じどうで はんていし、
  **達成すると バッジを獲得**して お祝い演出が出て、**つぎのミッションに 自動で
  きりかわり**ます。獲得した **バッジの一覧**（未獲得は 🔒 で ヒント表示）も
  見られます。いちど とったバッジは、あとで消えません。

- 🔍 **けんさく**
  あつめた虫を、**なまえ・📍みつけたばしょ・目・レア度**でしぼりこみ。
  みつけたばしょのタグは、図鑑に場所がふえるたびに 自動でふえていきます。
  目のタグは、あつめた虫にある目だけが本の順にならびます。

### こどもがたのしいしかけ

- 🎉 みつけたときの **紙吹雪** と **ファンファーレ音**
- 🔬 AIが「かんがえている」スキャン演出
- ⭐ レア度がわかる星ひょうか、レア虫（星5）はキラキラ
- 🏆 図鑑コンプリートでおいわい表示
- まるっこくてカラフルな、こども向けデザイン

## うごかしかた

```bash
npm install
npm run dev
```

ブラウザで表示される URL（例: `http://localhost:5173`）をひらいてください。
スマホやタブレットのカメラからそのまま撮影して読みこめます。

図鑑のデータはブラウザ（localStorage）にほぞんされるので、
とじても消えません。写真は保存まえに小さく圧縮（長辺512px・目安1枚22KB）
するので、**200件以上**ためられます。ふるい大きな写真は、つぎに開いたとき
自動で小さくしなおして、空きようりょうを ふやします。

## AIのモードについて

このアプリには2つの判定モードがあります。

### 1. デモモード（キーなしでも遊べる／かんたん）

写真の色を解析して、いちばん近い虫を図鑑からえらぶだけの「当てずっぽう」です。
本物の画像認識ではないので、正確ではありません（緑の虫はぜんぶ緑の虫に寄る等）。
すぐ試したいときや、あそび用です。

### 2. 本物AIモード（おすすめ・正確）

[Claude](https://www.anthropic.com/) の画像認識をつかい、写真の虫を**種レベルの和名**
（例: オオカマキリ、ギンヤンマ）まで正確に判定します。目・レア度・生息地・説明文も
その種に合ったものを返します。昆虫にくわしい子ども向けの精度をめざしています。

**つかいかた（公開サイトでもローカルでも同じ）:**

1. 右上の **⚙️ ボタン**（AIせってい）をひらく
2. [console.anthropic.com](https://console.anthropic.com/settings/keys) で作った
   Anthropic APIキー（`sk-ant-…`）を貼りつけて「ほぞんして有効化」

- キーは**その端末のブラウザ（localStorage）だけ**に保存され、どこにも送信・共有
  されません。リポジトリにも含まれません。
- ブラウザから直接 Anthropic API を呼ぶため、キーは自分専用にしてください。
- 使った分だけ Anthropic から料金がかかります。
- 本物AIモードで判定に失敗したときは、まちがった当てずっぽうを出さずにエラー表示します。

ローカル開発では `.env` に `VITE_ANTHROPIC_API_KEY=…` を書いて既定キーにすることも
できます（任意）。

### 3. AIチャット取り込みモード（APIキーなしでOK・手作業）

お手もちの **AIチャット（ChatGPT / Claude など）** をつかって同定し、その答えを
取り込めます。APIキーは不要で、追加のAPI料金もかかりません。

判定結果の画面で「🤖 AIチャットでもっと正確にしらべる」をひらき、

1. **① 写真と質問をAIチャットへ** … スマホの共有機能で、写真と決まった形式の質問文を
   AIチャットアプリに送ります（共有先に出ないアプリもあります。共有が使えない端末では
   質問文をコピーしてChatGPTを開きます）。
2. AIが `名前: / 目: / レア度: / 生息地: / 説明:` の形で、**コードブロック（```）に入れて**
   答えます。→ チャットアプリの**コピーボタンで全文をワンタップコピー**できます。
3. **貼り付け → ③ 取り込む** を押すと、各項目に自動入力されます（```の囲みが付いたままでもOK）。

※ ChatGPT Plusのような「チャットの月額課金」は、アプリから直接よび出すAPIとは別物です。
このモードは“手作業でAIチャットを使い、答えだけ取り込む”しくみなので、月額課金を活かせます。

## つかっている技術

- [Vite](https://vite.dev/) + [React](https://react.dev/) + TypeScript
- 図鑑の保存: ブラウザの localStorage
- 効果音: Web Audio API（音声ファイルなし）
- 画像なし・絵文字ベースなので、そのまま動きます
